## Python语言中随机数的生成基于随机数"种子"，每个种子作为输入，利用算法生成一系列随机数，构成伪随机序列。

- random.seed(s)
  - 作用：为随机数序列确定种子，其中参数为种子
  - 参数
    - s：随机数种子，一个整数或浮点数
- random.random([])
  - 作用：生成[0.0,1.0]之间的随机小数
- random.randint(a,b)
  - 作用：生成[a,b]之间的随机整数
- random.getrandbits(k)
  - 作用：生成k比特长度的随机整数，其中K是二机制位数的长度
  - 参数
    - K：一个整数
- random.randrange(start,stop[,step])
  - 作用：生成一个[start,stop]之间以step为步数的随机整数，使用方法与range()相似
  - 参数
    - start：一个整数，表示开始整数
    - stop：一个整数，表示结束整数
    - step：一个整数，表示步数
- random.uniform(a,b)
  - 作用：生成一个[a,b]之间的随机小数
  - 参数：
    - a、b：一个整数或浮点数
- random.choice(seq)
  - 作用：从序列类型中随机返回一个元素，序列类型包括列表、元祖和字符串
  - 参数
    - seq：一个序列类型变量
- random.shuffle(seq)
  - 作用：将序列类型中元素随机排列，返回打乱后的序列。序列类型包括列表。由于排序后对原有变量进行重写，所以该函数不能作用与不可变序列，即主要用于列表类型
- random.sample(pop,k)
  - 作用：从pop表示的组合数据类型中随机选取k个元素，以列表类型返回。注意，pop中所含的元素要不少于k个
  - 参数
    - seq：一个组合数据类型，如集合、列表、元祖、字符串等
    - k：一个整数
